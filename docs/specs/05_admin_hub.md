### 2.2 Метрики дашборда

| Метрика | Описание | Расчёт |
|---------|----------|--------|
| Выручка | Сумма оплаченных заказов | SUM(orders.total_amount) WHERE status != 'cancelled' |
| Заказы | Количество заказов | COUNT(orders) |
| Средний чек | Средняя сумма заказа | AVG(orders.total_amount) |
| Новые клиенты | Уникальные новые пользователи | COUNT(users) WHERE created_at >= period_start |
| Активные заказы | По статусам | COUNT по status: paid, accepted, preparing, ready |

## Пример

### Название
Система управления продуктами

### Описание

**Что:** CRUD API для управления продуктами в системе

**Зачем:** Позволить администраторам управлять каталогом продуктов

**Пользователь:** Администратор

### Поля/Данные

- `name` (string) - Название продукта
- `price` (number) - Цена
- `description` (string, опционально) - Описание
- `isActive` (boolean) - Активен ли продукт

### API Endpoints

- `POST /products` - Создать продукт
- `GET /products` - Список продуктов
- `GET /products/:id` - Детали продукта
- `PATCH /products/:id` - Обновить продукт
- `DELETE /products/:id` - Удалить продукт

### Приоритет
High

### Acceptance Criteria

- [ ] Можно создать продукт с name и price
- [ ] Можно получить список всех продуктов
- [ ] Можно получить детали продукта по ID
- [ ] Можно обновить продукт
- [ ] Можно удалить продукт
- [ ] Валидация полей работает
